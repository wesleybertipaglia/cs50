# Python - Data Types

```python
my_int = 5 # Integer
my_float = 5.5 # Float

my_true = True # Boolean
my_false = False # Boolean

my_str = "Hello, World!" # String

my_list = [1, 2, 3, 4, 5] # List
my_tuple = (1, 2, 3, 4, 5) # Tuple

my_dict = {"name": "John", "age": 30} # Dictionary
my_set = {1, 2, 3, 4, 5} # Set
```
